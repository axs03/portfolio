Some Basic MySQL Table commands :
Command Description
SELECT * FROM table_name; Retrieve all records from a table. [READ]
INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...); Insert a new record into a table. [CREATE]
UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition; Update records in a table based on a condition. [UPDATE]
DELETE FROM table_name WHERE condition; Delete records from a table based on a condition. [DELETE]

The operations above cover the basic CRUD functionality.
